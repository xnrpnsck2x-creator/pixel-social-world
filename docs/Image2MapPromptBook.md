# Image 2 Map Prompt Book

## Global Style Lock

Use this style lock for every official map prompt:

```text
Original top-down 2D pixel art map for a cozy fantasy social MMO, warm forest-town mood, readable early-2000s online RPG composition without copying any existing game, compact but clear town planning, rich plants and cozy props, small chibi avatar scale, stone paths and dirt paths that are easy to walk on, central gathering space, clear entrances, no UI, no text, no labels, no logos, no watermark, no characters in the foreground, no modern objects, crisp pixel edges, harmonious low-saturation palette, game-ready whole-map background.
```

## Negative Lock

```text
Do not include UI panels, readable words, Chinese/Japanese/English labels, logos, watermarks, giant foreground characters, photorealism, smooth vector art, 3D render look, noisy unreadable paths, blocked main roads, extreme darkness, exact copies of existing game maps, or text baked into signs.
```

## Registration Command

After each Image 2 output is selected, register it with:

```bash
python3 scripts/Tools/MapPipeline/register_generated_map.py <map_id> \
  --image /path/to/processed.png \
  --source /path/to/image2_source.png \
  --metadata-scaffold \
  --notes "Image 2 generated <short purpose> motherboard."
```

The script copies PNGs into `assets/maps/generated/`, upserts `configs/art_assets.json`, updates `configs/map_catalog.json`, and can create a conservative `configs/map_points.json` scaffold for first H5 patrol. Replace scaffolded walkable, NPC, portal, and blocked rectangles before exposing the map through a production route.

Risk gate: once a map has `asset_path`, `metadata_status: "scaffold"` is forbidden. Promote it to `playtest_metadata` with authored walkable, blocked, spawn, portal, gathering, and camera records before H5 patrol.
